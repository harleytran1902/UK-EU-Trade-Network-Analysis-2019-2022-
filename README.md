# UK - EU Trade Network Analysis (2019 - 2022)
Network analysis of UK-EU trade relationships using Python, Tableau, and Gephi

## 1. Project Summary



## 📌 Overview
This project analyzes how the UK–EU trade network changed after Brexit.

## 🎯 Problem
- Trade analysis usually only looks at volume  
- It does not show relationships between countries  

## ⚙️ Method
- Data cleaning  
- Network analysis  
- Visualization  

## 📈 Result
- Trade became more regional  
- UK lost some influence  

## 🛠 Tools
- Python  
- Excel  
- Gephi
